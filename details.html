Student guide

This page explains how the Erasmus Semester Planner can support a student from the first application steps to final departure preparation.

1. Create your account

Start with registration or login, then review your profile information, destination country and semester dates.

2. Build the semester plan

Use the dashboard to keep deadlines visible, watch your progress and track how many tasks still need attention.

3. Manage tasks and documents

Add tasks for flights, accommodation, learning agreement updates, insurance, coordinator meetings and application forms. Keep file names together in the document section.

4. Check reminders

Review announcements and checklist items regularly so you do not miss submission deadlines.
