LAB3 - Erasmus Semester Planner for VDU Students

Files included:
- index.html
- details.html
- styles.css
- script.js
- assets/images (5 visuals)
- assets/videos (2 mp4 demo clips)

How to upload to a server:
1. Unzip the LAB3 folder.
2. Upload all files and folders to your hosting/server public directory.
3. Open index.html in the browser, or set it as the default page.
4. If your teacher asks for a zipped version, upload LAB3.zip.

Note:
This is a frontend MVP prototype based on the LD2 document. It uses only HTML, CSS and JavaScript.
